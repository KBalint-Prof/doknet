## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## DÖKNet

LOGÓ:

Bagoly: A bagoly régóta a bölcsesség, tudás és tanulás szimbóluma. Mivel a DÖKNet egy diákönkormányzati oldal, ez a szimbólum hangsúlyozza az oktatási környezethez való tartozást és a tudás megszerzésének fontosságát. A bagoly szárnyai, vagy az azt körülölelő formák, dinamizmust és aktivitást sugároznak.

Körben nyilak / áramlás: A baglyot körülölelő nyilak, vagy egymásba fonódó ívek a folyamatos fejlődést, az események áramlását és a közösség dinamizmusát jelképezik. Ez utalhat a rendezvények szervezésére és az azokon való részvételre is. A körforma a teljességet, az egységet és a közösséget is erősíti.

Fogaskerék: A bagoly egyik szeménél látható fogaskerék az eseményszervezés, a működés, a logisztika és a hatékonyság jelképe. Ez azt sugallja, hogy a DÖKNet egy jól szervezett, működőképes platform.

Csillag: A bagoly másik szeménél lévő csillag a siker, a kitűnőség, az inspiráció és a diákok számára nyújtott lehetőségek szimbóluma. Utalhat a kiemelkedő rendezvényekre és az azokon szerzett élményekre.

Naptár: A bagoly testébe integrált naptár ikon egyértelműen a rendezvényszervezésre, időpontok kezelésére és a diákok értesítésére utal. Ez kiemeli a DÖKNet fő funkcióját, mint eseményplatform.

Bordó és további színek: A bordó szín a kifinomultságot, energiát és ambíciót képviseli, amely illeszkedik egy diákönkormányzati oldal dinamikus jellegéhez. A többi szín (kék, zöld, türkiz) frissességet, megbízhatóságot és fejlődést sugároz, melyek fiatalos és modern hangulatot kölcsönöznek a logónak.

Fokos sapka: A "DÖKNet" szóban az "e" betűn található fokos sapka ismét megerősíti a diákokhoz, az oktatáshoz és az érettségihez való kapcsolódást.
